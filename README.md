# Ecommerce - HackerU final project

Welcome to my E-Commerce Hacker-U project. This is a fullstack project with frontend and backend.

### In order to run server side:
`npm run server`

### On server side I used:
* Node.js
* mongoose
* dotend
* bcryptjs
* multer
* morgan
* express.js

### On client side I used:
* React
* Bootstrap
* CSS
* joi-browser
* axios
* React-Bootstrap
* toastify
